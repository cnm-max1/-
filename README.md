# 学生信息管理系统

纯前端 Web 应用：管理学生档案、五科成绩与班级考勤。数据存于浏览器 localStorage，无需后端。

**技术栈：** React 19 · TypeScript · Vite 6

## 快速开始

需要 Node.js 18+。

```bash
cd info-management-system
npm install
npm run dev
```

打开 http://localhost:5173 。构建：`npm run build` ，预览：`npm run preview` 。

> **Windows：** PowerShell 若无法运行 `npm`，改用 `npm.cmd`，或使用 cmd。
